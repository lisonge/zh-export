# zh-export

前端项目中文字符串批量识别导出工具

- 基于 AST 识别导出, 识别完整字符串节点
- 支持识别字符串变量拼接表达式

使用场景: 现有项目迁移至国际化, 初期需要快速批量导出每个文件的中文字符串

| 字符串表达式                                       | 结果                        |
| -------------------------------------------------- | --------------------------- |
| `'1个苹果'`                                | `1个苹果`                   |
| `` `1个苹果和${count}个香蕉` ``              | `1个苹果和{}个香蕉`         |
| `'1个苹果和'+count+'个香蕉'`             | `1个苹果和{}个香蕉`         |

## 使用

```sh
pnpm i
```

项目根目录新建 `folder.txt` 文件填入要导出的目录, 每行一个

```sh
pnpm start
```

在当前项目 `dist` 目录下会生成对应的导出 json 文件
